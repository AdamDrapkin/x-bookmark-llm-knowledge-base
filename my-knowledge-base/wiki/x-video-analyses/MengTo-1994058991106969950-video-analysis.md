## Summary

This is a comprehensive 45-minute tutorial by designer MengTo on how to prompt AI-generated animations using Gemini 3. The tutorial demonstrates creating polished landing pages with advanced animations including beam effects, clip animations, flashlight effects, and border gradients. It covers both the prompting techniques and the workflow using Aura as the primary design tool.

## Key Topics Covered

### 1. Animation Types Demonstrated
- **Beam animation** — Glowing border effect that follows cursor movement
- **Clip animation** — Text clipping effects for visual impact
- **Flashlight effect** — Mouse-following light effect on cards
- **Border gradient** — Gradient borders that follow the mouse
- **Scroll animations** — Animation triggered on scroll
- **Marquee animation** — Continuous scrolling carousel effects
- **Card rotation** — 3D card rotation between multiple cards
- **Mouse tracking** — Shader animations that follow mouse movement

### 2. Design Workflow
- Using [[aura]] as the primary design interface
- Collecting inspiration from [[mobbin]], [[dribbble]], [[mymind]]
- Breaking designs into individual sections rather than full pages
- Using [[unicorn-studio]] backgrounds and assets
- Exporting to [[cursor]], [[lovable]], [[ai-studio]]

### 3. Prompting Techniques
- Specific vs generic prompting differences
- Using reference code snippets for accuracy
- Extracting design systems as JSON
- Applying styles from one template to another
- The importance of being specific to avoid AI "hallucinations"
- Using "keep" and "important" keywords to preserve content

### 4. Advanced Techniques
- **Design system extraction**: Getting JSON format of fonts, colors, spacing, shadows, animations
- **Style transfer**: Applying exact styles from one template to another
- **Reference-based prompting**: Using code snippets for complex animations
- **Human-AI collaboration**: AI handles 90%, human adds final 10%

## Notable Techniques

1. **One-shot generation**: Getting quality results on first try with specific prompts
2. **Reprompting**: Going back to fix individual elements after initial generation
3. **Style JSON**: Extracting full design system as JSON for reuse
4. **Code snippet references**: Using actual code for complex animations rather than describing them
5. **Remixing**: Taking existing templates and modifying them
6. **Human touch**: Adding final 10% manually for professional results
