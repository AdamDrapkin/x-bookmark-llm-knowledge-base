# Video Analysis: MengTo - 1991489885690364187

**Source:** [Tweet](https://x.com/MengTo/status/1991489885690364187)

## Summary

This is a 37-minute tutorial by MengTo (founder of Aura and Gradient) on how to get the best results from Gemini 3 for web design. He demonstrates that while Gemini 3 is far more creative and better at animation than previous versions, it requires specific techniques to achieve optimal results.

## Key Topics Covered

1. **Inspiration & References**
   - Using tools like Mobbin, Dribbble, and Panda to collect design references
   - Feeding references, code, and images to Gemini 3
   - Using "adapt" to maintain consistency across sections

2. **Editing & Refinement**
   - Using design tools like Unicorn Studio to fix issues
   - Adding animations (border beams, interactions)
   - Changing icons, images, logos

3. **Design Assets & Templates**
   - Using Aura's templates and components
   - Backgrounds, buttons, and other assets
   - Exporting to platforms like Cursor, Lovable, V0, Figma

4. **Design Techniques**
   - Image-to-HTML conversion
   - Light and dark mode consistency
   - Typography and style control
   - Creating "liquid glass" effects with border gradients

## Notable Insights

- Gemini 3 is described as "the golden era for web design" (like mid-journey for web)
- Human involvement remains crucial - AI is a tool, not a replacement
- Single prompt iteration rarely produces perfect results - iteration is key
- The combination of AI generation + manual refinement creates the best outcomes
