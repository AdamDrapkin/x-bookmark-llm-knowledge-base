# Langchain

**Type:** Product/Tool
**Known For:** An open-source framework that enables developers to build applications powered by large language models by connecting them to external data sources and computation tools.

## Overview

LangChain is an open-source framework designed to simplify the creation of applications using large language models (LLMs). It provides tools for developers to chain together various components—including prompts, models, memory, and data sources—to build sophisticated AI-powered applications. The framework supports building chatbots, question-answering systems, and autonomous agents that can interact with external APIs and data sources. LangChain streamlines the development of LLM-powered applications by providing modular components for connecting language models to external data, APIs, and tools.
