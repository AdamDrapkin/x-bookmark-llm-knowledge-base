# Mobbin

**Type:** Product/Tool
**Known For:** Curated collection of mobile app UI designs and interaction patterns for designer inspiration

## Overview

Mobbin is a comprehensive design resource platform that aggregates screenshots and UI patterns from popular mobile applications. It allows designers to browse thousands of hand-picked designs organized by categories, components, and screens. The platform serves as a visual reference library for UI/UX designers seeking inspiration for their own projects.
