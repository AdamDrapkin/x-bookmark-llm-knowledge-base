# n8n

**n8n** (n8n.io) is an open-source workflow automation platform that enables users to connect APIs and services through a visual interface. It provides a powerful alternative to commercial automation tools with the flexibility of self-hosting.

## Key Features

- **Visual Workflow Builder** - Drag-and-drop interface for creating automations
- **Open Source** - Free to use, self-hostable
- **Extensive Integrations** - Connect to hundreds of services
- **Smart Alerting** - Filter noise and only notify on meaningful events

## Use Cases

- Data synchronization between apps
- Automated notifications (email, Telegram, Slack)
- Business process automation
- API orchestration
