# Perplexity

**Type:** AI Search Engine
**Founded:** 2022
**Headquarters:** San Francisco, California
**Known For:** AI-powered search, Deep Research mode, conversational answers

## Overview

Perplexity is an AI-powered search engine that provides direct answers to queries using large language models. Unlike traditional search engines that return links, Perplexity synthesizes information from multiple sources to provide comprehensive answers. Their "Deep Research" mode is specifically designed for comprehensive research tasks.

## Key Features

- **AI-Powered Search** — Conversational answers from web content
- **Deep Research** — Comprehensive research mode for complex topics
- **Source Citations** — Links to original sources for verification
- **Pro & Enterprise Plans** — Advanced features for professionals
