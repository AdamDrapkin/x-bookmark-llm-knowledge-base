# God of Prompt

**Type:** Content Creator / Prompt Engineer
**Platform:** X (Twitter)
**Known For:** AI prompt sharing, prompt engineering tutorials, prompt library products

@godofprompt is one of the most prominent accounts sharing prompt engineering techniques on X. They have built a large following by sharing practical prompts for ChatGPT, Claude, Gemini, and other AI tools. Their content includes frameworks, tutorials, and a commercial prompt library.
