# Cursor

**Type:** Product/Tool
**Known For:** An AI-powered code editor that uses artificial intelligence to assist developers with code completion, generation, and editing tasks.

## Overview

Cursor is an AI-first code editor built as a fork of Visual Studio Code, launched in 2023 by Anysphere, a company founded by former MIT students. It integrates large language models directly into the coding workflow, allowing developers to chat with AI about their codebase, auto-complete code snippets, and make intelligent refactoring suggestions. The tool supports pair programming with AI, where developers can collaborate with the AI assistant to write, debug, and improve code in real-time. Cursor has emerged as a leading AI-powered development environment, competing with traditional code editors by embedding generative AI capabilities directly into the editing experience to boost developer productivity.
