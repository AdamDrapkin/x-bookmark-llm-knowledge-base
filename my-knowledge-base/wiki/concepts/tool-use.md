# Tool Use

**Category:** Technique
**Definition:** The practice of AI systems utilizing external tools, APIs, and code execution capabilities to extend their problem-solving abilities beyond inherent knowledge and reasoning.

## Overview

Tool use enables AI agents to interact with external systems, execute code, query databases, and perform actions in real-world environments. This capability transforms static language models into dynamic agents capable of completing complex, multi-step tasks. The technique involves defining tool specifications, managing tool selection, and handling outputs returned to the model.
