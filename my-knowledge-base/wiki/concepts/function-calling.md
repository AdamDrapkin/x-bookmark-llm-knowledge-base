# Function Calling

**Category:** Technique
**Definition:** Function calling is a feature that enables an AI model to invoke external functions or APIs during response generation, allowing it to perform specific tasks or retrieve real‑time information beyond its internal knowledge.

## Overview

Function calling allows large language models to interact with external tools and services by invoking predefined functions during conversation. This extends their capabilities beyond static knowledge, enabling them to execute actions such as querying databases, making API calls, or controlling devices in real time. To use function calling, developers provide a set of function schemas, and the model decides when and which functions to call based on the user’s request. Function calling extends AI models by allowing them to execute external functions to complete tasks or retrieve up‑to‑date information during conversations.
