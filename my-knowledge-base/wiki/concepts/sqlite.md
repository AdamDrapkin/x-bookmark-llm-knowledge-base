# Sqlite

**Category:** Tool
**Definition:** A lightweight, serverless, self-contained relational database engine that stores data in a single file.

## Overview

SQLite is an embedded SQL database engine that requires no separate server process or configuration, making it ideal for mobile applications, browsers, embedded systems, and small-to-medium websites. It implements most of the SQL standard and uses a single disk file for all database contents.
