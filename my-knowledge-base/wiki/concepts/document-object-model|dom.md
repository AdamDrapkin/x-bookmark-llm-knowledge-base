# Document Object Model (DOM)

**Category:** Technique
**Definition:** The programming interface for web documents

## Overview

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. AI tools interact with the DOM to automate web interactions.
