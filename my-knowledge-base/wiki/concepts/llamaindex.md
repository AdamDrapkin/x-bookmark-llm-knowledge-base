# Llamaindex

**Category:** Tool
**Definition:** A data framework for building LLM applications that enables connection of custom data sources to large language models.

## Overview

LlamaIndex is an open-source framework designed to simplify the process of building applications powered by large language models. It provides tools for ingesting, structuring, and querying data from various sources, allowing developers to create context-augmented AI applications efficiently. The framework supports numerous data sources and offers flexible APIs for indexing and retrieval operations.
