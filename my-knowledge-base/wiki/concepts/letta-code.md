# Letta Code

**Category:** Tool
**Definition:** A memory persistence system designed to enable AI coding agents like Claude Code to retain context, preferences, and learned information across different sessions, addressing the problem of "session amnesia."

## Overview

Letta Code extends the capabilities of AI coding agents by implementing persistent memory layers that survive session boundaries. Unlike traditional AI coding setups that reset completely when a session ends, Letta Code allows agents to remember project-specific conventions, user preferences, and accumulated context from previous interactions. This creates a more coherent and efficient development experience, especially for long-term projects.
