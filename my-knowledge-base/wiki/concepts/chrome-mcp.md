# Chrome Mcp

**Category:** Tool
**Definition:** A browser extension or integration that brings Model Context Protocol (MCP) capabilities directly into the Google Chrome environment.

## Overview

Chrome MCP enables developers and users to interact with MCP servers and AI models directly from their Chrome browser. It provides a convenient interface for accessing context-aware AI assistance without leaving the browser, supporting tasks like web research, content analysis, and automated workflows. The tool integrates Chrome's browsing capabilities with MCP's standardized protocol for AI communication. Chrome MCP serves as a bridge between Chrome browser functionality and MCP-powered AI systems, allowing seamless access to context-aware AI capabilities while browsing the web.
