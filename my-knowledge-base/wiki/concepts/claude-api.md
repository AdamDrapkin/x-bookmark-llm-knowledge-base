# Claude Api

**Category:** Tool
**Definition:** Anthropic's API platform for integrating Claude's AI capabilities into applications, supported by official code recipes and command-line tools.

## Overview

Claude API enables developers to build applications powered by Claude, Anthropic's AI assistant. The Claude Cookbooks repository provides official Jupyter notebooks and code recipes demonstrating practical implementation patterns. Claude Code offers a command-line interface with 93 commands organized across 12 functional categories for seamless API interaction and project management.
