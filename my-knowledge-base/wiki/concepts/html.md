# Html

**Category:** Technique
**Definition:** Hypertext Markup Language, the standard markup language for creating web pages and web applications.

## Overview

Html provides the structural foundation of web content, using tags and elements to define headings, paragraphs, links, images, and other components. It works alongside CSS for styling and JavaScript for interactivity to create complete web experiences. Every website relies on Html as its core building block.
