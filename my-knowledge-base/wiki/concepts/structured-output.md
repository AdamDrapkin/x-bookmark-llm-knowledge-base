# Structured Output

**Category:** Technique
**Definition:** A prompt engineering technique that instructs AI systems to generate responses in specific, machine-readable formats rather than free-form text.

## Overview

Structured output is a technique used to ensure AI models produce responses that conform to predetermined schemas or formats, such as JSON, XML, or delimited text. This approach enables reliable parsing, integration with downstream systems, and consistent behavior across different queries. It is particularly valuable when building AI agents and applications that require programmatic access to model outputs.
