# Retrieval Augmented Generation

**Category:** Technique
**Definition:** A method that enhances language model outputs by retrieving relevant information from external knowledge bases before generating responses.

## Overview

Retrieval Augmented Generation (RAG) combines the power of large language models with real-time information retrieval from external sources. When a query is received, the system searches relevant documents or databases, then uses this retrieved context to inform the model's response generation. This approach grounds AI outputs in factual, up-to-date information while reducing hallucinations.
