# Nvidia A100

**Category:** Hardware
**Definition:** A high-performance data center GPU designed for AI, HPC, and analytics workloads, powered by NVIDIA's Ampere architecture.

## Overview

The NVIDIA A100 Tensor Core GPU delivers unprecedented acceleration for AI training and inference, data analytics, and scientific computing. It features third-generation Tensor Cores with support for FP16, BFLOAT16, TF32, and FP64 precision, enabling up to 20x performance improvements over previous generations. The A100 can be flexibly partitioned into seven independent instances to optimize resource utilization for different workloads.
