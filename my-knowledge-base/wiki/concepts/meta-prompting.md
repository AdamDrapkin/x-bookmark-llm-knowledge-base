# Meta Prompting

**Category:** [Technique]
**Definition:** Meta Prompting is a prompting methodology that uses a model to generate, refine, or optimize prompts for downstream tasks, essentially employing the model as its own prompt engineer.

## Overview

Meta Prompting leverages the model’s reasoning capabilities to construct effective input prompts, often by providing a high‑level instruction or template that guides the creation of task‑specific prompts. This approach automates the prompt design process, improves consistency, and enables rapid iteration on prompt performance without manual trial‑and‑error. It is especially useful when large numbers of customized prompts are needed or when domain expertise for prompt crafting is limited.
