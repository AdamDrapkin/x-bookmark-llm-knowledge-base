# Browser Automation

**Category:** Technique
**Definition:** Using AI to control web browsers for automated tasks

## Overview

Browser automation involves using AI agents to interact with web pages, fill forms, extract data, and perform repetitive web tasks. Tools like Anthropic's Computer Use capability enable AI models to control browsers directly.
