# Computer Use

**Category:** Technique
**Definition:** An AI agent capability enabling autonomous interaction with computers, browsers, and software applications to complete tasks without direct human input.

## Overview

Computer Use refers to the ability of AI systems to directly operate computing environments—clicking, typing, navigating interfaces, and executing software actions—to fulfill user requests. This technique represents a significant shift from AI responding with text to AI actively performing digital work. Providers like Perplexity have developed computer-use tools that position themselves as alternatives to existing solutions like OpenClaw, targeting professional workflows and automation scenarios.
