# MengTo's Gemini 3 UI Prompting Tutorial

**Author:** @MengTo
**Category:** design
**Date:** 2025-11-20
**Type:** standalone
**Source:** [Tweet](https://x.com/MengTo/status/1991489885690364187)

## Summary

MengTo, the designer behind [[Design Code]] and [[Spirit]] app, releases a comprehensive 37-minute tutorial on leveraging [[Gemini 3]] for creating high-quality UI designs. He positions the current moment as "the golden era for web design," drawing a parallel to the [[Midjourney]] era for visual design. The tutorial addresses his initial skepticism—his first attempts with generic prompts (like "create a landing page") produced the predictable "purple colors and generic type of UI" that most designers have come to expect from AI-generated interfaces. However, through strategic prompting techniques, he demonstrates how Gemini 3 can produce exceptional results with sophisticated animations, non-generic corner styling, and professional typography.

The tutorial is structured around three core techniques. First, **inspiration feeding**—collecting reference designs from curated sources like [[Mobbin]], [[Dribbble]], and [[MyMind]] to provide AI with concrete visual blueprints rather than abstract instructions. Second, **design editing**—using tools like [[Unicorn Studio]] and [[Aura]] to refine outputs, fix logos, add animations (including border beams and interactions), and enhance visual hierarchy. Third, **workflow integration**—exporting final designs to tools like [[Cursor]], [[Lovable]], and [[Figma]] for further development. MengTo emphasizes that specificity is paramount: rather than requesting a complete landing page, designers should prompt for individual sections with explicit style references ("in the style of Apple") to achieve professional monochrome colors, typography, and visual hierarchy. The result is an AI-assisted workflow where Gemini 3 handles creative generation and iteration, while specialized tools handle the granular refinement and export.

## Key Takeaways

- Gemini 3 significantly outperforms previous models in creativity, typography, animation, and logic for UI generation
- Generic prompts produce generic results—specificity and style references are essential for quality output
- Building a curated inspiration library from tools like Mobbin, Dribbble, and MyMind is foundational to effective AI design prompting
- Design references should be broken into individual sections (hero, animations, icons) rather than complete landing pages
- Gemini 3 excels at non-rounded corners, which [[GPT]] and [[Claude]] typically default to in their outputs
- The workflow combines Gemini 3 for generation, specialized tools for animation/editing, and Cursor for refined development control
- Remix capabilities allow combining designs into new styles (e.g., light mode → dark mode variations)
