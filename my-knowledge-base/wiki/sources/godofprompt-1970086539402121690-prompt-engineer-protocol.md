# This ChatGPT prompt will turn you into a prompt engineer

**Author:** @godofprompt
**Category:** prompt-engineering
**Date posted:** 2025-09-22
**Date ingested:** 2026-04-09
**Type:** bookmark

## Summary

This tweet from God of Prompt shares their "Protocol for Designing Prompts" - a systematic approach to creating effective AI prompts. The content is presented across two images showing the full methodology.

**Image 1** shows the "Protocol for Designing Prompts" with six key steps:
1. Define the Goal
2. Identify the Audience
3. Choose the Role
4. Set the Tone & Style
5. Build Examples (few-shot learning)
6. Test and Refine

**Image 2** outlines the "Golden Rule" - always ask: "Would this prompt give the best result for a non-expert user? If not, revise."

The content promotes godofprompt.ai as the source for more AI prompting resources.

## Key Takeaways

- Six-step protocol for designing prompts
- Few-shot learning through embedded examples
- Testing and refinement cycle
- Golden Rule: optimize for non-expert users
- Promotes godofprompt.ai for more resources
