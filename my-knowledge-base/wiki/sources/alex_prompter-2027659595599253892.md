# SkillsMP - Open-Source Agent Skills Marketplace for Claude Code

## Summary

SkillsMP is an emerging platform that functions as an "App Store for Claude Code," providing a centralized marketplace where developers can discover and install open-source agent skills. The platform hosts over 269,000 skills using the open SKILL.md standard, which enables skills to be discovered, shared, and installed across different agent frameworks. The marketplace supports both keyword-based and AI-powered semantic search, allowing users to find relevant skills through natural language queries like "skills about trading" or "data analysis related skills."

The platform provides multiple browsing options, including category navigation, popularity sorting, and recency tracking. A trend analytics graph demonstrates steady growth in skill creation since December 2025, indicating an active and expanding ecosystem. Both [[anthropic]] and [[langchain]] have officially integrated SkillsMP into their development tools—[[anthropic]] references the marketplace in their official Claude Code documentation as a resource for discovering community-built skills, while LangChain has added SkillsMP compatibility to their deepagent-CLI, enabling users to access the growing public skill collection.
