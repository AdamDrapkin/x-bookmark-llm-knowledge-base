## Summary

This tweet shares a meta-prompt technique that allows users to generate structured JSON prompts through natural language conversation. The core idea is deceptively simple: paste a system prompt into [[chatgpt]], then describe any task you want a prompt for, and the AI will return a properly structured JSON prompt template without additional explanation or commentary. The system prompt acts as a constraint layer, forcing the AI into a rigid output format that follows consistent rules—responding only in JSON, never adding explanatory text, and using placeholders when information is missing rather than guessing.

The technique addresses a common pain point in prompt engineering: writing well-structured prompts takes practice and iteration. This meta-prompt automates that process by creating a prompt generator that handles the structural work, allowing users to focus on describing their actual desired outcomes. The approach leverages [[meta-prompting]] principles—using prompts to generate better prompts—wrapped in a strict JSON output constraint that ensures consistency and machine-readable results.

## Key Takeaways

- Meta-prompting can automate prompt creation by describing desired outcomes instead of manually structuring prompts
- JSON output constraints enforce clean, consistent, and programmatically parseable results
- The "never guess" rule with placeholders prevents hallucination and keeps outputs predictable
- This approach is particularly useful for building prompt libraries or creating reusable prompt templates
- The technique shifts the user's cognitive load from "how do I write this prompt" to "what do I want this prompt to accomplish"
