## Summary

This tweet promotes a "JSON-only prompt generator" mega prompt designed to act as a reusable [[prompt-engineering]] tool. The core idea is simple: instead of manually writing prompts each time, users load this system prompt once and then describe any task in natural language, receiving a structured [[JSON]] output as a result.

The mega prompt works by operating as a middle layer between casual intent and structured prompt creation. When you say something like "write a prompt for this [add the command]," the system interprets your request and outputs a properly formatted JSON prompt that includes required fields. This enforces consistency across generated prompts and ensures no critical [[prompt structure|fields]] are missing.

The rules embedded in this mega prompt reveal important [[prompt-engineering]] principles: strict output formatting eliminates explanation and chatter; refusing to guess forces deliberate specification; required fields enforce comprehensive prompt architecture. This approach is particularly useful for users who want to build a [[prompt library]] or standardize their [[prompt workflow|workflow]] without writing every prompt from scratch. The trade-off is that it adds a layer of abstraction—users must work through the generator rather than getting direct results.

## Key Takeaways

- **Single Load, Multiple Use:** Load this system prompt once to generate unlimited structured prompts on demand without repeating instructions.
- **Structured Output Enforcement:** The generator outputs only valid JSON, eliminating explanations, apologies, or extra commentary from the AI.
- **Placeholder Instead of Guess:** Missing information triggers a placeholder field rather than the model making assumptions, preserving prompt accuracy.
- **Required Field Architecture:** Every generated prompt includes mandatory structural fields, enforcing [[prompt architecture]] best practices.
- **Natural Language Entry Point:** Users describe desired outcomes in conversational language, making prompt creation accessible to non-technical users.
