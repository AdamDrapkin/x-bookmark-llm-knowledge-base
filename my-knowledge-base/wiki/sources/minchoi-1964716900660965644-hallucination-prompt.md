## Summary

This bookmark points to a prompt that addresses the hallucination problem in AI models. The prompt functions as a "Reality Filter" - a directive scaffold that makes AI models more likely to admit when they don't know something rather than fabricating information. The bookmark includes an image showing the prompt text, which provides a mechanical approach to reducing hallucinations through repeated verification steps.

The concept is significant because hallucinations remain one of the biggest barriers to AI reliability in production contexts. By explicitly instructing the model to acknowledge uncertainty rather than generate plausible-sounding but incorrect information, users can get more accurate outputs especially for factual queries.

## Key Takeaways

- The "Reality Filter" prompt uses a directive scaffold approach to reduce hallucinations
- It works by making the model explicitly acknowledge uncertainty
- There are Google Gemini and Claude versions available as well
- The approach is "mechanical" - using repeated verification steps rather than hoping for better reasoning
