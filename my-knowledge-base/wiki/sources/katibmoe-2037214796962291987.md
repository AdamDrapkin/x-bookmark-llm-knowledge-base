# One: AI Agent Integration Platform

**Author:** @katibmoe
**Category:** ai-agents
**Date:** March 26, 2026
**Type:** product-announcement
**Source:** [Tweet](https://x.com/katibmoe/status/2037214796962291987)

## Summary

One is an AI agent integration platform that provides the simplest way to connect and monitor AI agents to hundreds of applications. The platform's main differentiator is the open-sourcing of what they claim to be the world's largest integration database, encompassing 47,000 agentic actions across more than 250 applications. The launch offer includes access to the platform plus 1 million free API requests per month, requiring users to retweet and comment "One" to qualify.

## Key Takeaways

- Platform enables connecting AI agents to hundreds of applications for monitoring and integration
- Claims the world's largest integration database with 47,000 agentic actions
- Coverage spans 250+ applications
- Launch offer: 1M free API requests/month with RT + comment requirement
- Open-source approach to sharing the integration database
