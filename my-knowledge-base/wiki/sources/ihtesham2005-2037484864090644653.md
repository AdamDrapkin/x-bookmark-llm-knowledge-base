## Summary

Claude Subconscious is an experimental plugin designed to solve the most significant limitation of Claude Code and similar AI coding agents: **session amnesia**. When you close a Claude Code session, it forgets everything—there's no persistent memory of your projects, patterns, or preferences across sessions.

Claude Subconscious addresses this by running as a background agent that watches everything Claude Code does, reads your files, builds up memory over time, and whispers hidden guidance back to the "main Claude" instance. This creates a "split brain" architecture where the primary agent receives continuous contextual guidance from a persistent memory layer.

The plugin leverages [[Letta]]'s memory system, tool access, and [[context engineering]] capabilities. It's worth noting that Letta also offers a direct alternative called [[Letta Code]]—a memory-first, model-agnostic, fully open-source coding agent designed specifically for this use case.

## Key Takeaways

- **Solves session amnesia**: Provides persistent memory across Claude Code sessions that would otherwise be lost
- **Background agent architecture**: Runs silently while Claude Code works, observing and building context
- **Split brain design**: Main Claude instance receives whispered guidance from the subconscious memory layer
- **Built on Letta**: Leverages Letta's established memory and context engineering systems
- **MIT licensed**: Fully open source for experimental use
- **Experimental status**: Acknowledged as a workaround to extend a closed-source/black-box agent
