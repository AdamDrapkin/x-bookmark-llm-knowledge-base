# Open Source AI Job-Hunting Agent Built for Claude Code

## Summary

Developer **@TBXark** created an autonomous AI agent designed to work within [[Claude Code]] that completely automates the job-hunting process end-to-end. The agent was built to scrape career sites, personalize CVs for each specific role, and automatically fill out application forms. After using the tool to submit over 700 applications, the developer successfully landed a job—and then made the entire project open source for others to benefit from.

This represents a significant advancement in AI-powered job automation, demonstrating how [[AI agents]] can handle tedious, repetitive tasks at scale while maintaining the personalization required to pass ATS (Applicant Tracking Systems) screening. The tool bridges the gap between mass application submission and quality, customized applications by leveraging [[Claude Code]]'s ability to understand job descriptions and translate them into tailored application materials.

The open-sourcing of this tool has sparked interest in the [[AI career tools]] space, particularly among developers and job seekers looking to leverage [[LLM]] capabilities for career advancement. It's part of a broader trend of [[agentic workflows]] being applied to real-world productivity challenges.

## Key Takeaways

- Built specifically as an [[agent]] for [[Claude Code]], leveraging its code execution and language understanding capabilities
- Automated the complete job application pipeline: job discovery, CV tailoring, and form submission
- Successfully used to submit 700+ applications before landing a job
- Released as open source, enabling community contribution and customization
- Demonstrates the practical application of [[AI agents]] to high-volume, repetitive real-world tasks
- Could potentially be adapted for other application types beyond job hunting
