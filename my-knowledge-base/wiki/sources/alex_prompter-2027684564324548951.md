## Summary

This tweet shares a meta-prompt designed to leverage a hidden feature in [[Claude]] called "SKILLS." According to the thread, SKILLS allow users to define reusable, production-grade workflows that the AI can execute repeatedly without needing to re-explain the context each time. The mega prompt positions Claude as an "expert SKILLS architect" that constructs these persistent automation templates through conversation.

The core insight is that instead of repeatedly providing the same context and instructions for recurring tasks, users can use this meta-prompt to architect a SKILL that encapsulates the workflow permanently. The tweet claims this is "the most powerful automation" the author has used, enabling [[one-shot learning]] of complex processes that can be executed forever afterward.

The visible portion of the prompt begins by assigning the role: *"You are an expert Claude SKILLS architect who creates reusable, production-grade skill prompts that turn one-time..."* (the prompt appears truncated in the source).
