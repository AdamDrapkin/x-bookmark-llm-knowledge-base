# The D-E-P-T-H Prompt Framework

**Author:** @godofprompt
**Category:** prompt-engineering
**Date posted:** 2025-10-03
**Date ingested:** 2026-04-09
**Type:** bookmark

## Summary

This tweet from God of Prompt introduces the D-E-P-T-H Prompt Framework, a comprehensive system for writing effective AI prompts. The framework consists of five key components:

1. **D - Define Multiple Perspectives**: Instead of using a single role, use multiple expert viewpoints collaborating. Example: "You are three experts: behavioral psychologist, copywriter, and data analyst."

2. **E - Establish Success Metrics**: Define specific, measurable outcomes upfront. Instead of "make it good" → "optimize for 40% open rate, 12% CTR, include 3 psychological triggers."

3. **P - Provide Context Layers**: Stack multiple context elements (business type, budget, audience, past performance) for more targeted output.

4. **T - Task Breakdown**: Break requests into sequential steps instead of one big ask. Creates logical flow and prevents confusion.

5. **H - Human Feedback Loop**: Build in self-critique. Example: "Rate your response 1-10 on clarity, persuasion, and actionability. Improve anything below 8."

## Key Takeaways

- Five-component framework for prompt optimization
- Multi-perspective approach prevents generic outputs
- Measurable metrics enable optimization
- Context layering improves relevance
- Sequential task breakdown prevents AI confusion
- Self-critique mechanism produces dramatically better results
