## Summary

This prompt template transforms [[gemini]] into a visual analysis specialist capable of dissecting any image into structured [[json]] output. The prompt instructs the model to act as an "expert visual analysis specialist with 15+ years" of experience, then processes an attached image to extract comprehensive metadata across multiple dimensions: composition (rule of thirds application, focal points, visual hierarchy), color profiles (dominant colors with hex codes and percentages), and lighting (source type, direction, quality).

The template demonstrates a powerful technique for [[prompt-engineering]]: instead of asking AI to describe an image loosely, this framework forces structured, quantifiable output. Each JSON field serves a specific purpose—color percentages sum to understanding overall palette dominance, focal points trace the viewer's eye path, and lighting details capture the technical conditions that make an image feel cohesive. This structured approach makes the output machine-readable and immediately useful for tasks like [[style-transfer]], image recreation, or building datasets for [[computer-vision]] training. The example outputs show analysis of what appears to be a lifestyle photograph (woman in a Porsche) with an achromatic color palette and bright red accent color.

## Key Takeaways

- **Structured output beats freeform description** — Forcing [[json]] format ensures consistent, comparable analysis across different images rather than variable-length descriptions
- **Role assignment enhances output quality** — The "15+ years of experience" framing primes the model to provide expert-level technical analysis
- **Quantifiable metrics make output actionable** — Hex codes, percentages, and directional descriptions (e.g., "90-degree side lighting") enable precise recreation
- **Composition analysis includes psychological elements** — The "visual hierarchy" field tracks how elements guide viewer attention through the frame
- **Multi-dimensional extraction** — The template covers metadata, composition, color, and lighting as four pillars of visual analysis

## Extracted Content

### Prompt Template

```
IMAGE TO JSON GENERATOR

[ATTACH AN IMAGE]

## Context & Goal
You are an expert visual analysis specialist with 15+ years of experience. Your task is to thoroughly analyze the provided image and convert your findings into a structured JSON format.

## Image Analysis
{
  "metadata": {
    "confidence_score": "high/medium/low",
    "image_type": "photograph/illustration/digital art/etc",
    "primary_purpose": "social media/advertising/art/etc"
  },
  "composition": {
    "rule_applied": "rule of thirds/centered/diagonal/etc",
    "aspect_ratio": "original ratio",
    "layout": "single subject/group/etc",
    "focal_points": [
      "Primary: [specific element]",
      "Secondary: [specific element]"
    ],
    "visual_hierarchy": "[how the eye moves through the image]",
    "balance": "symmetric/asymmetric - [explanation]"
  },
  "color_profile": {
    "dominant_colors": [
      {
        "color": "[name]",
        "hex": "[hex code]",
        "percentage": "[amount]",
        "role": "[purpose in image]"
      }
    ],
    "color_palette": "[palette description]",
    "temperature": "warm/neutral/cool",
    "saturation": "high/moderate/low",
    "contrast": "high/medium/low"
  },
  "lighting": {
    "type": "natural/artificial/studio/etc",
    "source_count": "[number of sources]",
    "direction": "[directional description]",
    "directionality": "hard/soft/diffused",
    "quality": "[additional quality notes]"
  }
}
```
