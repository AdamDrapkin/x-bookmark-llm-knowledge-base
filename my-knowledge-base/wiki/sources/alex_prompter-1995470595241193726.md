# Image to JSON Generator Prompt

**Author:** @alex_prompter

**Date:** 2025-12-01
**Type:** standalone
**Source:** [Tweet](https://x.com/alex_prompter/status/1995470595241193726)

## Summary

This tweet shares a [[Gemini]] prompt template designed to transform any image into a structured [[JSON]] output containing comprehensive visual analysis. The prompt establishes the AI as an "expert visual analysis specialist" and instructs it to generate detailed breakdowns across multiple technical dimensions including metadata, composition, color profiles, and lighting conditions. The author presents this as a practical utility for prompt engineers who need to extract precise technical information from reference images for use in [[AI image generation]] workflows.

The resulting JSON structure organizes visual analysis into nested categories: metadata (confidence scores, image type, primary purpose), composition (rule of thirds application, aspect ratio, focal points, visual hierarchy, balance), color profile (dominant colors with hex codes and percentages, palette type, temperature, saturation, contrast), and lighting (type, source count, direction, quality). This systematic approach transforms subjective visual observations into machine-readable data that can be fed into [[text-to-image]] models or used to reverse-engineer specific visual styles.

## Key Takeaways

- **Structured output format**: The prompt generates consistent, machine-readable JSON that can be easily integrated into downstream AI pipelines or used as input for image generation models.
- **Technical depth**: Rather than simple descriptions, the output captures specific technical attributes like hex color codes, lighting direction, and composition rules applied.
- **Visual hierarchy analysis**: Includes tracking of eye movement patterns and focal point prioritization — crucial for understanding what makes an image effective.
- **Color role assignment**: Each dominant color is tagged with a specific role (primary subject, background, accent) making the output actionable for style replication.
- **Multi-domain applicability**: The prompt works across image types (photographs, illustrations, social media content) and serves various use cases from style transfer to prompt engineering.

## Extracted Content

### Prompt Template

```
IMAGE TO JSON GENERATOR

[ATTACH AN IMAGE]

## Context & Goal
You are an expert visual analysis specialist with 15+ years of experience in...

## Image Analysis
{
  "metadata": {
    "confidence_score": "high",
    "image_type": "photograph",
    "primary_purpose": "social media"
  },
  "composition": {
    "rule_applied": "rule of thirds",
    "aspect_ratio": "1:1 (square)",
    "layout": "single subject",
    "focal_points": [
      "Primary: The woman's face, specifically her sunglasses and profile.",
      "Secondary: The red seatbelt and the Porsche steering wheel."
    ],
    "visual_hierarchy": "The eye is first drawn to the woman on the left, then follows the strong diagonal of the red seatbelt...",
    "balance": "asymmetric - The main subject occupies the left two-thirds of the frame, balanced by..."
  },
  "color_profile": {
    "dominant_colors": [
      {
        "color": "Black",
        "hex": "#1B1A1D",
        "percentage": "50",
        "role": "primary subject"
      },
      {
        "color": "Bright Red",
        "hex": "#D81E2C",
        "percentage": "5",
        "role": "accent"
      }
    ],
    "color_palette": "achromatic with a strong accent",
    "temperature": "neutral",
    "saturation": "moderate",
    "contrast": "medium contrast"
  },
  "lighting": {
    "type": "natural window",
    "source_count": "multiple sources",
    "direction": "90-degree side",
    "directionality": "diffused",
    "quality": "soft lighting"
  }
}
```

### JSON Output Schema

| Category | Fields |
|----------|--------|
| **metadata** | confidence_score, image_type, primary_purpose |
| **composition** | rule_applied, aspect_ratio, layout, focal_points[], visual_hierarchy, balance |
| **color_profile** | dominant_colors[], color_palette, temperature, saturation, contrast |
| **lighting** | type, source_count, direction, directionality, quality |
