# Insanely Fast Whisper: 2.5 Hours Audio Transcribed in 98 Seconds

## Summary

Insanely Fast Whisper represents a significant breakthrough in on-device audio transcription, achieving what was previously only possible with cloud-based services. This open-source CLI tool combines OpenAI's [[whisper]] Large v3 model with cutting-edge optimization techniques including [[flash-attention-2]], FP16 precision, and intelligent batching to deliver transcription speeds that make cloud APIs largely unnecessary for most use cases.

The benchmark results on an Nvidia A100 80GB GPU demonstrate the dramatic impact of progressive optimization. The baseline Whisper Large v3 with Transformers at full FP32 precision takes approximately 31 minutes to process 150 minutes of audio. By applying FP16 quantization, batching with a batch size of 24, and BetterTransformer optimizations, this drops to just over 5 minutes. The final optimization layer—adding Flash Attention 2—brings the time down to an astonishing 1 minute and 38 seconds, representing roughly an 18x speedup from baseline.

For users seeking an even faster option, the Distil-Whisper variant (distil-large-v2) with Flash Attention 2 achieves the fastest result at just 1 minute 18 seconds, while still maintaining reasonable accuracy. This performance enables real-time or near-real-time transcription workflows that were previously impractical, all running entirely offline on local hardware without any API keys, subscriptions, or data leaving the user's machine.

## Key Takeaways

- Transcribes 150 minutes (2.5 hours) of audio in under 98 seconds using Whisper Large v3 on Nvidia A100 80GB
- Runs entirely on local GPU with no cloud dependencies, API keys, or subscription requirements
- Installation via a single command: `pipx install insanely-fast-whisper==0.0.15 --force`
- Flash Attention 2 provides the most significant performance boost, reducing transcription time by ~3x over standard configurations
- FP16 precision reduces memory footprint and computation by approximately 50% with minimal accuracy loss
- Batching with size 24 enables efficient GPU utilization across multiple audio segments simultaneously
- Distil-Whisper (distil-large-v2) offers the fastest option at 1 minute 18 seconds for scenarios where maximum speed is prioritized over absolute accuracy
- Built on Transformers library with Optimum and flash-attn for production-grade reliability
