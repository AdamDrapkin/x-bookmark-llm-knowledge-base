## Content

This is an n8n workflow template for **website downtime monitoring with smart alerts**.

### Key Features:
- Waits 5 minutes before alerting (filters out temporary hiccups)
- Sends alerts via email OR Telegram
- Full user control over notification settings

### Use Case:
Most monitoring tools alert on every tiny hiccup. This workflow solves that problem by waiting 5 minutes, which helps cut down on unnecessary notifications while still catching real issues.
