## 
What Are Connectors?

Lovable Connectors are pre-configured integrations that live in your Lovable project. Click, authenticate, and you're connected. No environment variables to manage, no server-side setup required.

## 
Shared Connectors

These connectors work for everyone using your app. Set them up once, and all users can access the integration.

Supabase - PostgreSQL database with real-time subscriptions, auth, and storage. Perfect for most SaaS apps.

Xano - No-code backend with visual database builder and API generation. Great for non-technical founders.

Airtable - Spreadsheet-database hybrid. Ideal for CMS, admin panels, or lightweight data needs.

## 
Personal Connectors

These connectors authenticate per user. Each person connects their own account, enabling personalized integrations.

Google Drive - Access user files, create documents, manage folders. Users authorize with their Google account.

Slack - Post messages, read channels, manage workspaces. Each user connects their Slack workspace.

GitHub - Read repos, create issues, manage code. Users authenticate with their GitHub account.

## 
APIs

Connect to any REST API with built-in authentication and request management.

- Stripe - Payment processing and subscriptions

- OpenAI/Anthropic - AI chatbots and content generation

- Firecrawl - Web scraping and data extraction

- Perplexity - AI-powered search and research

- Twilio - SMS notifications, voice calls, and WhatsApp messaging

- Telegram - Bot-based messaging, alerts, and community broadcasts

- Slack - Team alerts, channel updates, and workflow notifications

- Twitch - Stream overlays, live chat, follower tracking, and creator dashboards

- Replicate - AI image generation

- Pinecone - Vector search for AI applications

- DeepL - Translation services

- Cal.com - Calendar scheduling

## 
How to Use Them

1. Add connector in Lovable

1. Enter credentials (API keys, tokens, URLs)

1. Start making requests

That's it. No backend deployment, no environment variables, no configuration files.

## 
Why Connectors Beat Manual Integration

Speed: Set up in seconds vs. hours of configuration Security: Credentials stored securely, never in your code Simplicity: No backend deployment needed Flexibility: Switch providers without rewriting code

## 
Best Practices

- Use Shared Connectors for app-wide data (databases, CMS)

- Use Personal Connectors for user-specific integrations (their Google Drive, their Slack)

- Use Custom APIs for specialized third-party services



## Examples for Lovable Connectors

Shared Connectors (App-Wide Data)

- SaaS User Dashboard - Supabase to store user accounts, subscriptions, and activity logs

- Content Management - Airtable as a CMS for blog posts, product catalogs, or documentation

- Customer Database - Xano to manage customer records with visual workflows

- Multi-tenant App - Supabase with Row Level Security for isolated customer data

- Real-time Chat - Supabase real-time subscriptions for messaging apps

- Analytics Dashboard - Query Supabase for metrics, charts, and reporting

Personal Connectors (User-Specific)

- Document Generator - Users create reports in their own Google Drive

- Team Notifications - Send alerts to each user's Slack workspace

- Code Review Tool - Access user's GitHub repos and pull requests

- File Uploader - Let users save directly to their Google Drive

- Workflow Automation - Trigger actions in user's connected Slack channels

- Personal CRM - Sync contact data from user's Google account

Custom APIs

- Payment Processing - Stripe API for subscriptions and checkout

- Email Campaigns - SendGrid/Mailchimp for transactional emails

- SMS Notifications - Twilio for alerts and 2FA codes

- AI Features - OpenAI API for chatbots or content generation

- Maps & Location - Google Maps API for store locators

- Weather Data - Weather API for location-based features

- Social Media - Twitter/LinkedIn APIs for posting content

- Calendar Booking - Cal.com API for scheduling

Combo Use Cases

- Invoice Generator - Supabase for data + Google Drive connector to save PDFs to user's folder

- Project Management Tool - Supabase database + Slack notifications + GitHub integration

- Marketing Platform - Airtable for content + SendGrid API for emails + Stripe for billing

- Support Ticketing - Supabase for tickets + Slack for team alerts + Twilio for SMS updates
