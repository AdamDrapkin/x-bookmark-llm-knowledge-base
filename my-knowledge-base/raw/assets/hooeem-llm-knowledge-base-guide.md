# How to create your own LLM knowledge bases today (full course)

**Source:** [Tweet](https://x.com/hooeem/status/2041196025906418094) by @hooeem
**Related to:** Andrej Karpathy's LLM Knowledge Bases system

## Key Premise

> "If you're able to learn how to create an LLM knowledge base then you have essentially created your own 'external brain' that you can utilise..."

### The Problem with Typical AI Usage
- Most people use AI like a search engine with amnesia
- You ask a question, get an answer, close the tab
- Tomorrow you start from scratch
- Nothing accumulates, nothing compounds
- You're burning tokens to rediscover the same context over and over

### Karpathy's System Solution
1. **Collect raw material** — Articles, papers, YouTube transcripts, PDFs
2. **AI reads everything** — Writes a structured wiki with summaries, concepts, connections
3. **Ask questions** — Against the wiki, gets cited, synthesised answers
4. **File back** — Every answer gets filed back into the wiki
5. **Health-check** — AI periodically finds contradictions, gaps, outdated info, fixes them

## Three Versions Covered

### 1. YOUR FIRST KNOWLEDGE BASE (Complete Beginner)
- **Tools:** Obsidian (free) + Claude Pro ($20/month)
- **Steps:**
  1. Create vault in Obsidian
  2. Create two folders: `raw` and `wiki`
  3. Add 3-5 source articles to raw
  4. Ask AI to compile wiki
  5. Use Graph View to explore

### 2. THE FULL SYSTEM (Comfortable with AI)
- More automation
- Health-check routines
- Continuous accumulation

### 3. BUILDER/DEVELOPER
- Full system with developer customization

## Core Workflow Components

1. **Raw collection** — Source material storage
2. **AI compilation** — Structured wiki generation
3. **Query interface** — Question answering against wiki
4. **Feedback loop** — Answers filed back into wiki
5. **Health check** — Periodic maintenance (find contradictions, gaps, outdated info)

## Notes from the Article

- Works for "literally any topic: Crypto markets, medical research, legal case law, competitive intelligence, academic study, philosophy"
- After a month of feeding: "you have a deeply interlinked resource that no Google search could replicate"
- "Because it's been synthesised, not just indexed"
